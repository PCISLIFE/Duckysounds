Duck Click Quacker is a Chromium browser extension that plays random duck sounds when you click anywhere on a webpage. When a quack triggers, a small duck pops up on the page too.

## Features
- Random duck quacks on mouse clicks (not every click)
- Small duck visual appears when a sound triggers
- Works across all websites
- Enable/disable toggle in the popup
- Lightweight, no tracking

## Installation
1. Clone or download this repository
2. Open `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the extension folder
